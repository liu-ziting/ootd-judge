# OOTD JUDGE - AI 毒舌穿搭审判

一个基于 Vue 3 + TypeScript 的智能穿搭评价应用，通过 AI 技术对用户的穿搭进行毒舌式评价和专业建议。

## 🎯 项目简介

OOTD JUDGE 是一个有趣的穿搭评价应用，结合了：

-   **毒舌模式**：幽默犀利的吐槽式评价
-   **导师模式**：专业温和的改进建议
-   **AI 智能分析**：基于智谱 AI 的视觉分析能力
-   **双模式切换**：满足不同用户需求

## ✨ 核心功能

### 📸 图片获取

-   **实时摄像头**：支持前后摄像头切换，自动镜像处理
-   **图片上传**：支持从相册选择图片
-   **智能预览**：图片自动适配显示

### 🤖 AI 智能分析

-   **智谱 AI 集成**：使用 GLM-4.1V-Thinking-Flash 模型
-   **双重评价模式**：
    -   **毒舌模式**：幽默犀利的吐槽，评分从 A+ 到 F
    -   **导师模式**：专业详细的改进建议
-   **备用数据**：API 不可用时自动使用预设数据

### 🎨 用户体验

-   **动态分析动画**：扫描网格 + 进度条效果
-   **打字机效果**：逐字显示评价内容
-   **响应式设计**：适配移动端和桌面端
-   **状态管理**：清晰的拍照 → 分析 → 结果流程

## 🛠️ 技术栈

-   **前端框架**：Vue 3 (Composition API)
-   **开发语言**：TypeScript
-   **构建工具**：Vite
-   **样式方案**：原生 CSS3 + CSS 变量
-   **AI 服务**：智谱 AI (GLM-4.1V-Thinking-Flash)

## 🚀 快速开始

### 环境要求

-   Node.js 16+
-   现代浏览器（支持摄像头 API）

### 安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### AI 配置

1. 复制 `.env.example` 文件为 `.env`
2. 在智谱 AI 平台获取 API Key
3. 在 `.env` 文件中配置：

```
VITE_AI_API_KEY=your_zhipu_api_key_here
```

## 🎮 使用说明

### 基本流程

1. **选择图片源**：使用摄像头拍照或上传图片
2. **开始分析**：点击快门按钮开始 AI 分析
3. **查看结果**：
    - 毒舌模式：幽默犀利的吐槽评价
    - 导师模式：专业的改进建议（点击求教按钮）
4. **重新开始**：点击"再来一次"清空数据重新开始

### 功能特性

-   **摄像头切换**：前后摄像头自由切换
-   **图片预览**：上传图片实时预览
-   **状态重置**：再来一次时自动清空所有数据
-   **离线备用**：AI 服务不可用时使用预设数据
